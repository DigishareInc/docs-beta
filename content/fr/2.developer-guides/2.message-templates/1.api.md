Le endpoint `GET /message_templates` vous permet de récupérer la liste de vos modèles de messages (templates) configurés dans votre compte Digishare.

## 1. Lister les Modèles

::api-playground
---
method: GET
url: "https://api.digishare.ma/v1/message_templates"
headers:
  Authorization: "Bearer VOTRE_TOKEN"
variables:
  company_id: "VOTRE_COMPANY_ID"
responseSample:
  data:
    - id: "mtpl_wn65opmdnal84r7b"
      company_id: "comp_780ew4kx3kdyjpa5"
      message_type_id: "mtyp_r79wdekbalz5jn48"
      code: "welcome_customer_promo_v3"
      title: "WhatsApp - Welcome New Customer with Promo (Version 3)"
      template:
        name: "welcome_customer_promo_v3"
        language: "en_US"
        components:
          - type: "HEADER"
            format: "TEXT"
            text: "Welcome to Our Brand, {{1}}!"
          - type: "BODY"
            text: "Hello {{1}}, thank you for joining us! Use promo code {{2}} for 10% off."
  meta:
    pagination:
      total: 53
      count: 25
      per_page: 25
      current_page: 1
      total_pages: 3
---
::

### Paramètres

| Paramètre | Type | Requis | Description |
| :--- | :--- | :--- | :--- |
| `company_id` | String | **Oui** | ID du compte entreprise Digishare |
| `message_type_id` | String | Non | Filtrer par type de message (ex: WhatsApp HSM) |
| `is_used` | Integer | Non | `1` = actif, `0` = archivé |
| `search` | String | Non | Recherche par nom ou code du template |
| `page` | Integer | Non | Numéro de page (défaut: 1) |
| `limit` | Integer | Non | Éléments par page (défaut: 25, max: 100) |
| `sort_by` | String | Non | Champ de tri (ex: `-updated_at`, `title`, `code`) |

---

## 3. L'Objet Template

Chaque modèle contient un objet `template` décrivant la structure du message :

| Champ | Description |
| :--- | :--- |
| `template.name` | Nom du template (identique au code) |
| `template.language` | Langue du template (ex: `en_US`, `fr`) |
| `template.components` | Tableau des composants (HEADER, BODY, FOOTER, BUTTONS) |

### Types de composants

| Type | Description |
| :--- | :--- |
| `HEADER` | En-tête : peut être TEXT, IMAGE, VIDEO ou DOCUMENT |
| `BODY` | Corps du message avec variables `{{1}}`, `{{2}}`, etc. |
| `FOOTER` | Pied de page (texte court) |
| `BUTTONS` | Boutons d'action (Quick Reply, URL, Phone, etc.) |

::tip
Utilisez le `id` du modèle retourné comme valeur pour `message_template_id` lors de la [création de campagnes](/fr/developer-guides/campaigns).
::
