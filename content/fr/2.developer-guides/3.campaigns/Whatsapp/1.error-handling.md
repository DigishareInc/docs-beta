## Erreurs Courantes

### ❌ Modèle de message introuvable
L'identifiant `message_template_id` envoyé ne correspond à aucun modèle (**Texte**, **ITM** ou **WABA TM**) configuré dans votre Dashboard Digishare.

### ❌ Numéro de téléphone invalide
Le numéro de téléphone dans `wa_id` doit être au format international sans le signe `+` (ex: `212600000000`).

### ❌ Support média inaccessible
WhatsApp n'a pas pu télécharger le média. L'URL doit être publique, ne pas expirer rapidement, et le fichier ne doit pas dépasser les limites de taille (16 Mo pour la vidéo).

### ❌ Variable manquante
Toutes les variables définies lors de la création du modèle local (ex: `{{1}}`, `{{name}}`) doivent être fournies dans le payload API.

::warning
Les erreurs de modèle asynchrones (ex: rejet par Meta lors de l'envoi effectif) seront notifiées via vos Webhooks configurés.
::
