Ce guide démontre comment orchestrer une campagne WhatsApp présentant des **images de haute fidélité**.

::note
**Prérequis** :
- Un **Bearer Token** valide dans l'en-tête Authorization.
- Un **Modèle de Message Digishare** (ITM ou **WABA TM**) avec un en-tête image activé.
- Une URL d'image accessible publiquement (JPG ou PNG).
::

## 1. Exigences de la Charge Utile

Pour les images personnalisées, fournissez le lien de l'en-tête et les informations du destinataire dans les blocs `global_data` et `recipients`.

### Champs Requis
- `wa_id` : Numéro de téléphone du destinataire (dans `global_data` comme variable de modèle).
- `header.link` : URL publique directe de l'image (JPG ou PNG).
- `tags` : Tableau optionnel pour catégoriser les campagnes.
- `conversation_id` : Identifiant optionnel pour le suivi des conversations.

## 2. Implémentation API

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Envoyez des images promotionnelles uniques à chaque destinataire.
variables:
  token: VOTRE_TOKEN
  senderId: VOTRE_SENDER_ID
  templateId: VOTRE_ID_MODELE
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Campagne Promotionnelle Produit
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  global_data:
    wa_id: "%recipient.wa_id%"
    header:
      link: "https://votre-site.ma/produit-123.jpg"
  recipients:
    - wa_id: "212600000000"
    - wa_id: "212600000001"
  tags: []
  conversation_id: "message-out"
---
::

## 3. Bonnes Pratiques

- **Accessibilité** : Assurez-vous que les URLs de vos images sont accessibles publiquement sans authentification.
- **Ratio** : Un **format 1:1** est recommandé pour un rendu optimal dans les listes de conversations et carrousels WhatsApp.
