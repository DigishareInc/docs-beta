Ce guide détaille le processus d'envoi de **messages vidéo** de haute fidélité via WhatsApp.

::note
**Prérequis** :
- Un **Bearer Token** valide.
- Un **Modèle de Message Digishare** (ITM ou **WABA TM**) avec support d'en-tête vidéo activé.
::

## 1. Exigences de la Charge Utile

Pour les vidéos personnalisées, vous devez mapper la variable `header_url` dans le bloc `global_data` pour garantir une livraison correcte pour chaque destinataire.

### Spécifications Techniques
- `wa_id` : Numéro de téléphone du destinataire.
- `header_url` : URL publique directe vers le fichier MP4.
- **Taille Maximale** : 16 Mo.

## 2. Implémentation API

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Envoyez une démonstration vidéo personnalisée à vos destinataires.
variables:
  token: VOTRE_TOKEN
  senderId: VOTRE_ID_EXPEDITEUR
  templateId: VOTRE_ID_MODELE
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Campagne de Démo Produit
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  global_data:
    header_url: "%recipient.header_url%"
  recipients:
    - wa_id: "212600000000"
      header_url: "https://votre-site.ma/demonstration.mp4"
---
::

::warning
Assurez-vous que votre hébergeur vidéo supporte un grand nombre de téléchargements simultanés et que l'URL est accessible publiquement.
::
