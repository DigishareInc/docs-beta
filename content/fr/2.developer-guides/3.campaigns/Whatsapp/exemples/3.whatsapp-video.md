Ce guide détaille le processus d'envoi de **messages vidéo** de haute fidélité via WhatsApp.

::note
**Prérequis** :
- Un **Bearer Token** valide dans l'en-tête Authorization.
- Un **Modèle de Message Digishare** (ITM ou **WABA TM**) avec prise en charge des en-têtes vidéo activée.
- Une URL de vidéo MP4 accessible publiquement (16 MB maximum).
::

## 1. Exigences de la Charge Utile

Pour les vidéos personnalisées, fournissez le lien de l'en-tête et les informations du destinataire dans les blocs `global_data` et `recipients`.

### Attributs Techniques
- `wa_id` : Numéro de téléphone du destinataire (dans `global_data` comme variable de modèle).
- `header.link` : URL publique directe vers le fichier MP4.
- `tags` : Tableau optionnel pour catégoriser les campagnes.
- `conversation_id` : Identifiant optionnel pour le suivi des conversations.

## 2. Implémentation API

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Envoyez une démonstration vidéo personnalisée à vos destinataires.
variables:
  token: VOTRE_TOKEN
  senderId: VOTRE_ID_EXPEDITEUR
  templateId: VOTRE_ID_MODELE
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Campagne de Démo Produit
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  global_data:
    wa_id: "%recipient.wa_id%"
    header:
      link: "https://votre-site.ma/demonstration.mp4"
  recipients:
    - wa_id: "212600000000"
  tags: []
  conversation_id: "message-out"
---
::

::warning
Assurez-vous que votre hébergeur vidéo supporte un grand nombre de téléchargements simultanés et que l'URL est accessible publiquement.
::
