Ce guide détaille l'orchestration de campagnes WhatsApp incluant des **Carrousels**, un format haute fidélité supportant jusqu'à 10 cartes coulissantes interactives.

::note
**Prérequis** :
- Un **Bearer Token** valide dans l'en-tête Authorization.
- Un **Modèle de Message Digishare** (type **WABA TM**) approuvé pour l'orchestration de Carrousels.
- Des URL d'images/vidéos accessibles publiquement pour les cartes du carrousel.
::

## 1. Gouvernance du Carrousel

Chaque carte supporte des médias uniques (image/vidéo) et des variables de corps spécifiques pour une personnalisation profonde.

### Attributs Techniques
- `header: { link }` : URL d'actif visuel (image/vidéo) pour chaque carte spécifique.
- `body_variables` : Tableau localisé de valeurs pour l'injection de variables spécifiques à la carte.
- `tags` : Tableau optionnel pour catégoriser les campagnes.
- `conversation_id` : Identifiant optionnel pour le suivi des conversations.

## 2. Implémentation API

Pour les carrousels dynamiques, fournissez le tableau `cards` avec des liens médias personnalisés pour chaque destinataire via le bloc `global_data`.

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Envoyez un carrousel produit personnalisé avec des cartes uniques par destinataire.
variables:
  token: VOTRE_TOKEN
  senderId: VOTRE_ID_EXPEDITEUR
  templateId: VOTRE_ID_MODELE
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Showcase Produits
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  global_data:
    wa_id: "%recipient.wa_id%"
    cards: "%recipient.cards%"
  recipients:
    - wa_id: "212600000000"
      cards:
        - header:
            link: "https://site.ma/p1.jpg"
          body_variables: ["Vastux Pro", "199 USD"]
        - header:
            link: "https://site.ma/p2.jpg"
          body_variables: ["Vastux Air", "99 USD"]
  tags: []
  conversation_id: "message-out"
---
::

::important
L'orchestration de carrousels est exclusivement disponible pour les modèles **WABA TM** de catégorie **Marketing**.
::
