Ce guide démontre comment orchestrer des campagnes WhatsApp avec des boutons **Call-to-Action (CTA)** d'appel téléphonique pour un réengagement client immédiat.

::note
**Prérequis** :
- Un **Bearer Token** valide.
- Un **Modèle de Message Digishare** (ITM ou **WABA TM**) avec un bouton d'appel actif.
::

## 1. Logique Opérationnelle

::tip
Les numéros de téléphone sont généralement configurés de manière **statique** lors de la phase de gouvernance du modèle dans votre Dashboard Digishare, garantissant la cohérence architecturale.
::

## 2. Implémentation API

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Envoyez une campagne présentant un CTA d'appel téléphonique intégré.
variables:
  token: VOTRE_TOKEN
  senderId: VOTRE_ID_EXPEDITEUR
  templateId: VOTRE_ID_MODELE
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Contact Support
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  recipients:
    - wa_id: "212600000000"
---
::
