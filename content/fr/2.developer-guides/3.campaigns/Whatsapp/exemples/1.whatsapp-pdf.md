Ce guide explique comment envoyer une campagne WhatsApp incluant un **document PDF**.

::note
**Prérequis** :
- Un **Bearer Token** valide dans l'en-tête Authorization.
- Un **Modèle de Message Digishare** (ITM ou **WABA TM**) configuré pour accepter un en-tête avec lien PDF.
- Une URL PDF accessible publiquement.
::

## 1. Exigences de la Charge Utile

Pour envoyer un PDF personnalisé, vous devez fournir le lien de l'en-tête et les informations du destinataire via les blocs `global_data` et `recipients`.

### Champs Requis
- `wa_id` : Numéro de téléphone du destinataire (dans `global_data` comme variable de modèle).
- `header.link` : URL accessible du PDF.
- `tags` : Tableau optionnel pour catégoriser les campagnes.
- `conversation_id` : Identifiant optionnel pour le suivi des conversations.

## 2. Implémentation API

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Envoyez un document PDF personnalisé à plusieurs destinataires.
variables:
  token: VOTRE_TOKEN
  senderId: VOTRE_SENDER_ID
  templateId: VOTRE_ID_MODELE
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Distribution de Factures
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  global_data:
    wa_id: "%recipient.wa_id%"
    header:
      link: "https://votre-site.com/fact_janvier.pdf"
  recipients:
    - wa_id: "212600000000"
    - wa_id: "212600000001"
  tags:
    - "campagne"
  conversation_id: "message-out"
---
::

## 3. Gestion des PDF Statiques

Si votre campagne utilise le même PDF pour tous les destinataires, définissez le lien de l'en-tête une seule fois dans le bloc `global_data` :

```json
{
  "title": "Catalogue Mensuel",
  "channel": "whatsapp",
  "sender_label_id": "your-sender-label-id",
  "message_template_id": "your-message-template-id",
  "global_data": { 
    "wa_id": "%recipient.wa_id%",
    "header": {
      "link": "https://votre-site.com/catalogue.pdf"
    }
  },
  "recipients": [
    { "wa_id": "212600000000" },
    { "wa_id": "212600000001" }
  ],
  "tags": ["campagne"]
}
```
