Ce guide montre comment envoyer une campagne WhatsApp incluant un **bouton d'appel téléphonique**.

::note
**Prérequis** :
- **Jeton d'accès (Bearer Token)** valide.
- **Modèle de Message Digishare** (type ITM ou **WABA TM**) contenant un bouton d'appel.
::

## 1. Fonctionnement

::tip
Le numéro de téléphone est généralement configuré de manière **statique** lors de la création du modèle dans votre Dashboard Digishare. Il ne change pas d'un destinataire à l'autre.
::

## 2. Structure du Payload

Même si le bouton est statique, vous devez envoyer la campagne en spécifiant l'ID du modèle correct.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Envoyer un bouton d'appel." :variables='{"token": "VOTRE_TOKEN", "senderId": "VOTRE_ID_EXPEDITEUR", "templateId": "VOTRE_ID_MODELE"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Contact Support", "channel": "whatsapp", "sender_label_id": "{senderId}", "message_template_id": "{templateId}", "recipients": [{"wa_id": "212600000000"}] }'}
::

::warning
Certains types de boutons ne peuvent pas être combinés dans le même modèle. Pour les limitations, consultez les [Règles des Boutons](/fr/developer-guides/message-templates/whatsapp-button-rules).
::
