The `GET /message_templates` endpoint allows you to retrieve the list of message templates configured in your Digishare account.

## 1. List Templates

::api-playground
---
method: GET
url: "https://api.digishare.ma/v1/message_templates"
headers:
  Authorization: "Bearer YOUR_TOKEN"
variables:
  company_id: "YOUR_COMPANY_ID"
responseSample:
  data:
    - id: "mtpl_wn65opmdnal84r7b"
      company_id: "comp_780ew4kx3kdyjpa5"
      message_type_id: "mtyp_r79wdekbalz5jn48"
      code: "welcome_customer_promo_v3"
      title: "WhatsApp - Welcome New Customer with Promo (Version 3)"
      template:
        name: "welcome_customer_promo_v3"
        language: "en_US"
        components:
          - type: "HEADER"
            format: "TEXT"
            text: "Welcome to Our Brand, {{1}}!"
          - type: "BODY"
            text: "Hello {{1}}, thank you for joining us! Use promo code {{2}} for 10% off."
  meta:
    pagination:
      total: 53
      count: 25
      per_page: 25
      current_page: 1
      total_pages: 3
---
::

### Parameters

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `company_id` | String | **Yes** | Digishare company account ID |
| `message_type_id` | String | No | Filter by message type (e.g., WhatsApp HSM) |
| `is_used` | Integer | No | `1` = active, `0` = archived |
| `search` | String | No | Search by template name or code |
| `page` | Integer | No | Page number (default: 1) |
| `limit` | Integer | No | Items per page (default: 25, max: 100) |
| `sort_by` | String | No | Sort field (e.g., `-updated_at`, `title`, `code`) |

---

## 3. Template Object

Each template contains a `template` object describing the message structure:

| Field | Description |
| :--- | :--- |
| `template.name` | Template name (same as code) |
| `template.language` | Template language (e.g., `en_US`, `fr`) |
| `template.components` | Array of components (HEADER, BODY, FOOTER, BUTTONS) |

### Component Types

| Type | Description |
| :--- | :--- |
| `HEADER` | Header: can be TEXT, IMAGE, VIDEO or DOCUMENT |
| `BODY` | Message body with variables `{{1}}`, `{{2}}`, etc. |
| `FOOTER` | Footer (short text) |
| `BUTTONS` | Action buttons (Quick Reply, URL, Phone, etc.) |

::tip
Use the returned template `id` as the `message_template_id` value when [creating campaigns](/en/developer-guides/campaigns).
::
