This guide shows how to send a WhatsApp campaign including a **phone call button**.

::note
**Prerequisites**:
- Valid **Bearer Token**.
- **Digishare Message Template** (ITM or **WABA TM**) containing a call button.
::

## 1. How it Works

::tip
The phone number is usually configured **statically** during template creation in your Digishare Dashboard. It does not change from one recipient to another.
::

## 2. Payload Structure

Even if the button is static, you must send the campaign by specifying the correct template ID.

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Send a call button." :variables='{"token": "YOUR_TOKEN", "senderId": "YOUR_SENDER_ID", "templateId": "YOUR_TEMPLATE_ID"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "Contact Support", "channel": "whatsapp", "sender_label_id": "{senderId}", "message_template_id": "{templateId}", "recipients": [{"wa_id": "212600000000"}] }'}
::

::warning
Certain button types cannot be combined in the same template. For limitations, see the [Button Rules](/en/developer-guides/message-templates/whatsapp-button-rules).
::
