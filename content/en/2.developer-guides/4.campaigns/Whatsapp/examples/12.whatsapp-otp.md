This guide shows how to send a WhatsApp campaign including an **OTP code** (One-Time Password).

::note
**Prerequisites**:
- Valid **Bearer Token**.
- **Digishare Message Template** (**WABA TM** type) configured for authentication.
::

## 1. OTP Functionality

For WhatsApp OTPs, you must use an **Authentication** type template (WABA TM) to benefit from auto-complete features on Android/iOS.

### Required Fields
- `otp_code`: The verification code generated by your system.

## 2. Payload Example

::api-playground{method="POST" url="https://api.digishare.ma/v1/campaigns" description="Send an OTP code." :variables='{"token": "YOUR_TOKEN", "senderId": "YOUR_SENDER_ID", "templateId": "YOUR_TEMPLATE_ID"}' :headers='{"Authorization": "Bearer {token}", "Content-Type": "application/json"}' :body='{"title": "OTP Verification", "channel": "whatsapp", "sender_label_id": "{senderId}", "message_template_id": "{templateId}", "recipients": [{"wa_id": "212600000000", "otp_code": "456789"}] }'}
::

::important
The `otp_code` field is reserved for official WABA TM authentication templates. If you are using a simple text template, use a standard variable (`{{1}}`).
::
