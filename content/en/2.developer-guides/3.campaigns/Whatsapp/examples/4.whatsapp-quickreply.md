This guide demonstrates how to orchestrate WhatsApp campaigns featuring **Quick Reply** buttons for seamless customer interaction.

::note
**Prerequisites**:
- Valid **Bearer Token**.
- **Digishare Message Template** (ITM or **WABA TM**) configured with interactive button components.
::

## 1. Operational Logic

Quick reply buttons allow users to engage with your brand in a single tap. Each interaction triggers a real-time event captured by our [Webhook Orchestration](/en/developer-guides/webhooks) system.

## 2. API Implementation

::api-playground
---
method: POST
url: https://api.digishare.ma/v1/campaigns
description: Send a campaign with interactive Quick Reply orchestration.
variables:
  token: YOUR_TOKEN
  senderId: YOUR_SENDER_ID
  templateId: YOUR_TEMPLATE_ID
headers:
  Authorization: Bearer {token}
  Content-Type: application/json
body:
  title: Feedback Survey
  channel: whatsapp
  sender_label_id: "{senderId}"
  message_template_id: "{templateId}"
  recipients:
    - wa_id: "212600000000"
---
::

::tip
Interactive buttons significantly improve engagement rates for surveys, appointment confirmations, and service menus.
::
