## 1. Digishare Message Template Types

Digishare supports three types of templates for your campaigns:
- **Text**: Simple text message without complex tags.
- **ITM**: Interactive Template Message (buttons, lists).
- **WABA TM**: WhatsApp Business API Template (Meta-approved marketing/notifications).

::tip
For marketing and high-volume campaigns, we recommend using **WABA TM** templates to ensure maximum deliverability and compliance with Meta's policies.
::

## 2. Dynamic Variables

Place shared variables in `global_data` and unique values in each `recipients` object.

- **Dynamic Mapping**: Use `%recipient.variableName%` in `global_data` to generate complex attributes from individual recipient data.
- **Consistency**: Ensure every variable defined in the template (e.g., `{{1}}`, `{{name}}`) has a corresponding value.

## 3. Rate Limits

::warning
Respect sending limits to avoid your number being blocked by WhatsApp.
::

- **Scaling**: Start with small volumes to build your sending reputation (Meta Tiering).
- **Efficiency**: Digishare handles queuing, but you should prioritize bulk sending via the `recipients` array (up to 1,000 per call) rather than many individual API calls.

## 4. Template Maintenance

- **Approval**: Only templates with `APPROVED` or finalized status in the dashboard can be sent.
- **Modifications**: Any change to a **WABA TM** template on Meta requires a new approval period before it can be used via the Digishare API.
