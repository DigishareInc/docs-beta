## Comprehensive WhatsApp API Error Guide

This document serves as the master reference for all WhatsApp error codes (Cloud API, On-Premises, and 360dialog Gateway) as integrated into the system.

### 🚀 Quick Reference (Common Errors)

| Code | Label | Meaning | Action |
| :--- | :--- | :--- | :--- |
| **131049** | `ecosystem_limit` | Per-user marketing limit meta-imposed. | **Stop.** Wait 24h before retrying. |
| **131047** | `chat_expired` | Window > 24h since last user reply. | Send a **Template** message. |
| **131026** | `undeliverable` | Blocked, Not on WA, or restricted. | Verify number or ask user to unblock. |
| **2001** | `template_missing` | Template name/lang not found/approved. | Check spelling vs WhatsApp Manager. |
| **429** | `rate_limited` | Too many requests. | Implement **Exponential Backoff**. |

## 🌩️ Meta Cloud API Errors (13XXXX)

| Code | Description | Detailed Logic / Solution |
| :--- | :--- | :--- |
| **131021** | Self-Messaging | Sender and recipient phone number is the same. Test with a different number. |
| **131031** | Account Locked | Restricted for policy violations or 2FA mismatch. Disable 2FA, re-register, re-enable. |
| **131042** | Payment Issue | Overdue invoices, suspended WABA, or missing currency/timezone settings. |
| **131052** | Media Download Error | Unable to download media from the provided URL. Check MIME type and URL accessibility. |
| **131053** | Media Upload Error | Unsupported media type or server-side upload failure during message creation. |
| **131057** | Maintenance Mode | WABA is undergoing a throughput upgrade or maintenance. |
| **132000** | Param Count Mismatch | Variables in request != variables defined in the approved template. |
| **132012** | Param Format Mismatch | Variable value (e.g. date/currency) format doesn't match the template schema. |
| **135000** | Generic User Error | Related to template restrictions or security concerns (Meta specific). |

## 🏗️ On-Premises & Template Errors (1XXX / 2XXX)

| Code | Description | Key Insight |
| :--- | :--- | :--- |
| **1001** | Message Too Long | Text body exceeds the character limit (usually 4096 chars). |
| **1005** | Access Denied | API key lacks permission or registration is locked/verification failed. |
| **1006** | Resource Not Found | Recipient or specific media object not found on WA servers. |
| **2005** | Translated Text Too Long | Variables + Template text exceeds UI limits in certain languages. |
| **2007** | Policy Violated | Content violates Meta's character or formatting policies. |
| **2061** | Template Paused | High report rate from users. Lower quality rating resulted in a temporary pause. |
| **2062** | Template Disabled | Permanently disabled due to repeated low quality or policy breach. |

## 🌐 360dialog Infrastructure Errors

| Code | Message | Potential Solution |

## 🛠️ Developer Best Practices

1. **Exponential Backoff**: For **429** and **500** errors, implement a retry strategy (1s, 2s, 4s, 16s...).
2. **Template Quality**: Monitor quality scores for templates to avoid **2061 (Paused)**.
3. **24-Hour Rule**: Always check if a conversation window is open before sending free-form text (**131047**).
