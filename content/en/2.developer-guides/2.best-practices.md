Follow these recommendations to ensure the reliability and efficiency of your communications.

## 1. Media Management

::important
Media URLs (PDF, Image, Video) used in your messages must be **public and accessible**.
::

- **Accessibility**: Test your URLs in an incognito browser window to verify that no session IDs or cookies are required.
- **Supported Formats**: 
    - **Video**: MP4 only, maximum 16 MB.
    - **Image**: JPG, PNG (1:1 ratio recommended for carousels).
    - **Document**: PDF.

## 2. Dynamic Variables

Always use the `%recipient.variableName%` format in your global `data` object.

- **Consistency**: Ensure every variable defined in the template has a corresponding value in the `contacts` list.
- **Default Values**: If a variable is optional in your business logic, send an empty string rather than omitting it to avoid Meta parsing errors.

## 3. Rate Limits

::warning
Respect WhatsApp Business API sending limits.
::

- **Scaling**: Start with low volumes to build your sending reputation (Meta Tiering).
- **Parallelization**: Digishare handles queuing, but avoid sending thousands of individual requests simultaneously; prefer bulk sending via the `contacts` array.

## 4. Template Maintenance

- **Approval**: Only templates with `APPROVED` status in your Meta dashboard can be sent.
- **Modifications**: Any change to an existing template on Meta requires a new approval period before it can be used via the API.
