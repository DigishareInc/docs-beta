---
title: Reply Button
description: Send interactive reply buttons for quick responses.
icon: i-mdi-send
---


Best for quick confirmations (Yes/No). Maximum 3 buttons.

::api-playground
---
method: POST
url: "https://api.digishare.ma/v1/messages"
headers:
  Authorization: "Bearer YOUR_TOKEN"
  Content-Type: "application/json"
body:
  send_to_third: true
  conversation_id: "CONV_123"
  type: "interactive"
  body:
    type: "button"
    header:
      type: "text"
      text: "Confirmation"
    body:
      text: "Do you confirm receipt?"
    action:
      buttons:
        - type: "reply"
          reply:
            id: "btn_yes"
            title: "Yes"
        - type: "reply"
          reply:
            id: "btn_no"
            title: "No"
responseSample:
  id: "msg_999"
  status: "sent"
---
::
