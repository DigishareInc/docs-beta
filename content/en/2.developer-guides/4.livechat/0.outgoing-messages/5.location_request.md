---
title: Location Request
description: Request the user to share their location.
icon: i-mdi-send
---



Triggers the native "Send Location" interface.

::api-playground
---
method: POST
url: "https://api.digishare.ma/v1/messages"
headers:
  Authorization: "Bearer YOUR_TOKEN"
  Content-Type: "application/json"
body:
  send_to_third: true
  conversation_id: "CONV_123"
  type: "interactive"
  body:
    type: "location_request_message"
    body:
      text: "Please share your current location."
    action:
      name: "send_location"
responseSample:
  id: "msg_202"
  status: "sent"
---
::
