Send a message from an Agent (system user) to the contact in a conversation.

**Endpoint**: `POST https://api-app.digishare.ma/v1/event/conversation_message`

### Message Parameters / Payload

| Field                      | Type    | Description                                                                              |
| :------------------------- | :------ | :--------------------------------------------------------------------------------------- |
| `event_type`               | String  | Should be set to `conversation_message`.                                                 |
| `channel`                  | String  | The messaging channel (e.g. `whatsapp`).                                                 |
| `type`                     | String  | The type of the message (e.g. `text`).                                                   |
| `send_to_third`            | Boolean | Set to `true` to send the message to the external platform provider.                     |
| `api_provider_instance_id` | String  | The required ID of the API provider instance.                                            |
| `recipient_id`             | String  | The specific identifier of the recipient.                                                |
| `body`                     | String  | The actual content of the message.                                                       |
| `conversation_id`          | String  | Unique ID for the conversation *(Optional)*.                                             |

::warning
**24-Hour Guard**: If the elapsed time since the last inbound message exceeds 24 hours (e.g., WhatsApp's customer service window), delivery may be restricted by your provider.
::
